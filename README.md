# Dynamics365Enum
Dynamics 365 Enumeration Tool specialized enumeration script for Dynamics 365 penetration testing and red teaming.
Key Features:

-Endpoint Discovery: Enumerates Core, Debug, and API paths.

-Solution Fingerprinting: Identifies installed modules like Field Service, Project Service, and Voice of the Customer via static asset detection.

-Smart Calibration: Auto-detects and filters "Soft 404" / default error pages.

-Stealth & Control: Includes WAF evasion (request delays), proxy support, and customizable status code/size filtering.

-h for usage manual.
